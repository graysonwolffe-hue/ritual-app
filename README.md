# Ritual – Habit Tracker

## Deploy to Vercel (5 minutes)

### Step 1 — Install dependencies
Open Terminal in this folder and run:
```
npm install
```

### Step 2 — Test locally (optional)
```
npm run dev
```
Open http://localhost:5173 to preview.

### Step 3 — Deploy to Vercel
1. Go to https://vercel.com and sign up free (use GitHub login)
2. Click **"Add New Project"**
3. Click **"Import"** → then drag this whole folder onto the page
   - OR install Vercel CLI: `npm i -g vercel` then run `vercel` in this folder
4. Click **Deploy** — done in ~60 seconds
5. You'll get a URL like `ritual-abc123.vercel.app`

### Step 4 — Install on iPhone
1. Open your Vercel URL in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (box with arrow pointing up)
3. Scroll down → tap **"Add to Home Screen"**
4. Name it "Ritual" → tap **Add**
5. It's now on your home screen — launches fullscreen like a native app 🔥

## Your data
All habit data saves to your phone's local storage automatically.
It persists between sessions and works offline after first load.
